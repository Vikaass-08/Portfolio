## Getting Started

To run this portfolio on your local machine, follow these steps:

1. Clone this repository:

   ```bash
   git clone https://github.com/Vikaass-08/Portfolio.git
   ```
   
2. Install the dependencies:

   ```bash
   npm install
   ```
   
3. run Locally:

   ```bash
   npm run start
   ```
